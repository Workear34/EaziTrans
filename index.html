帮助

基本使用

将需要翻译的文本粘贴在文本框内，选择好对应的语言（或使用实验性的自动选择功能，不支持交换语言功能）之后即可自动开始翻译。

可以复制译文，也可以交换两种语言选项。

设置说明

API 地址：用来访问提供商 API 的 URL 地址。此选项为必填项，请到相应的 API 服务提供商的文档查询地址。随意更改此选项会导致无法正常翻译。

API 密钥：用来访问提供商 API 的密钥。此选项为必填项，如您没有 API 密钥，请到相应的 API 服务提供商进行申请。请注意：本网页使用的 LocalStorage 存储并不安全，因此请务必保管好您的 API 密钥。

模型选择：翻译时要使用的大模型。列表里的选项均为比较小巧且免费的大模型，未来将会支持自定义大模型功能。请注意不要使用推理模型，这会导致翻译速度非常慢，因为模型将会在每次翻译时都进行推理。

System Prompt：翻译时以 System 身份发给模型的 Prompt，用于指定模型扮演的角色。随意更改此选项会导致无法正常翻译。

User Prompt：翻译时以 User 身份发给模型的 Prompt，用于指定模型完成的任务。随意更改此选项会导致无法正常翻译。
